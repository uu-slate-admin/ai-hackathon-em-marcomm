# Runtime Assets

This directory holds assets used directly by the game runtime.

- `maps/`: campus maps, collision layers, and scene overlays
- `photos/`: location imagery used in dialogue cards, title art, and stop content
- `sprites/`: player, `Swoop`, items, and landmark sprite assets
- `typography/`: runtime font files
- `audio/`: ambient loops, effects, and UI sounds when added

Keep source-of-truth brand reference assets under `SKILLS/uofu-enrollment-branding/assets/`, not here.
